# MikroTik Whisperer

MikroTik router management dashboard powered by Next.js and Supabase.

## Quick Start

```sh
npm install
npm run dev
```

The development server starts at http://localhost:8080.

## Tech Stack

- Next.js (App Router)
- TypeScript
- React
- Supabase (Auth, Database, Edge Functions)
- shadcn/ui
- Tailwind CSS
- TanStack React Query
